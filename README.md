# Tax Calculator 2025 — Canada & Quebec

Free tax calculator covering Quebec and all 13 Canadian provinces/territories. Verified 2025 tax data. No signup required.

> Built by [Mark Systems](https://github.com/AILabManager-tech/mark-systems).

## Features

- Federal + provincial tax calculation for all provinces
- Dedicated Quebec page (QPP, solidarity credit, specific contributions)
- Up-to-date 2025 tax brackets
- Credits and deductions (RRSP, QPP, etc.)
- Full SEO (sitemap, robots.txt, OG, JSON-LD, hreflang)
- Static site — zero dependencies, zero framework

## Pages

| Page | Description |
|------|-------------|
| `index.html` | All-provinces calculator (EN) |
| `quebec.html` | Detailed Quebec calculator |
| `toutes-provinces.html` | Multi-province comparison view |

## Tech Stack

| Layer | Technology |
|-------|------------|
| HTML/CSS/JS | Vanilla (no framework) |
| Deployment | Vercel |
| SEO | XML sitemap, robots.txt, Open Graph, hreflang |
| Security | Vercel headers (vercel.json) |

## Getting Started

Open `index.html` in a browser. No build step required.
